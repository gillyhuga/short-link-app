## Getting Started

### Prerequisites

- [Node >= 16.14.0](https://nodejs.org/en/)

### Installation

- Clone repository
  ```
     https://github.com/gillyhuga/short-link-app.git
  ```
- Go to the project directory
  ```
     cd short-link-app
  ```
- Install dependencies

  ```
     npm install
  ```

- Start the server
  ```
     npm start
  ```
- Open `http://localhost:3000` with your browser to see the result
